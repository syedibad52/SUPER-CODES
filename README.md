# 🔐 Password Generator (Python)

A simple Python project that generates a strong random password using letters, numbers, and special characters.

---

## 🚀 Features
- Generates secure random passwords  
- Uses uppercase and lowercase letters  
- Includes digits and special characters  
- User-defined password length  

---

## 🛠️ Technologies Used
- Python 3  
- Built-in modules: `random`, `string`
